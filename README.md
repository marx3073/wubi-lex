# wubi-lex-tool
Win10自带的微软五笔码表安装、管理工具。 
不需要重启操作系统就可以切换词库，例如在五笔86、五笔98词库间自由切换。
支持导入、导出、安装二进制格式、文本格式的码表，支持字根反查。
